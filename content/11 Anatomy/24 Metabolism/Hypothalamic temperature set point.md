set point for activation is around 37.1°C:
- too low → shivering
- too high → sweating 

## Skin Temperature
Skin temp can alter set point:
ROT: ↓ skin temp → ↑ set point vice versa
### Heat response
higher skin temp → need lose more heat → lower set point → sweat earlier

### Cold response
lower skin temp → need produce more heat → higher set point → shiver earlier
hairs on skin also undergo piloerection, but quite minimal contribution as human v little skin hair
