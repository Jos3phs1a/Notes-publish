## Types of contraction

| Contraction | Description                                                                                                                                                                                                                                                                                                                                                                                           |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Isotonic    | load constant, muscle length change                                                                                                                                                                                                                                                                                                                                                                   |
| Isometric   | muscle length constant, tension change<br>Force w/o muscle length change<br><br>Internal torque = external torque                                                                                                                                                                                                                                                                                     |
| Isokinetic  | velocity remain, muscle length change                                                                                                                                                                                                                                                                                                                                                                 |
| Concentric  | Force via muscle shortening<br><br>Tinternal > Texternal<br><br>[[Muscle Tension Properties\|Force-velocity relationship]]<br>		Heavier load, slower you can lift(vel ↓)<br>		Most energy is released as heat (75%) remaining is work done                                                                                                                                                            |
| Eccentric   | Force via muscle lengthening<br>induces most muscle growth<br>$T_{internal}$ < $T_{external}$                                                                                                                                                                                                                                                                                                         |
| Note        | Factor in agonist antagonist muscles,<br>Speed of action determine which muscle is actively working<br>e.g. Elbow flexion at 90$\degree$ shoulder flexion<br> <br>After 90$\degree$ elbow flexion, If speed of forearm falling:<br>= free fall: passive, done by gravity, no work by bicep/triceps<br><br>> free fall: bicep working concentrically<br><br>< free fall: triceps working eccentrically |
