All muscles contain mix of different muscle types depending on function

|            | Slow | Fast |
| ---------- | ---- | ---- |
| Oxidative  | 1    | 2a   |
| Glycolytic |      | 2x   |
Fast twitch fibers have **Large** [[motor units]] while slow twitch fibers have **small** [[motor units]] 

Speed: time taken for peak tension
	==Slow: ~50ms==
	==Fast: ~15ms==
	Rate of ==Myosin- ATPase activity== → how fast ATP is converted to be used for crossbridge cycle

Oxidative vs glycolytic:  Amt of mitochondria → oxidative phosphorylation capacity
	Oxidative → aerobic respiration → ==resist fatigue better==
	Glycolytic → anaerobic

Meat more ==red → ↑ myoglobin,== ↓type 1, ↑type 2x
