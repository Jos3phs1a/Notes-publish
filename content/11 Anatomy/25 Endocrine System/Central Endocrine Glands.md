## Hypothalamus
### Functions
connects CNS to endocrine system
- coordinate with Posterior pituitary to secrete ADH & oxytocin, ==hypothalamus produces, post pit regulates secretion==
- secrete hypothalamic releasing or inhibiting hormones to act on anterior pituitary gland
	- difference hypo + post pit → secrete  vs hypo controls ant pit secretion
- control sympathetic output - neuroendocrine reflex

### Hypothalamic Control
#### to anterior pituitary
Hypothalamus secretes hormones to anterior pituitary
connects via [[hypophyseal portal system]]


| Hypophysiotropic Hormones                                                               | Actions                                                |
| --------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| Growth Hormone Releasing Hormone (GHRH)<br><br>Growth Hormone Inhibiting Hormone (GHIH) | Stimulates GH release <br><br>Inhibits GH, TSH release |
| Thyrotropin - releasing Hormone (TRH)                                                   | Stimulates TSH, PRL rlease                             |
| Corticotropin - releasing Hormone (CRH)                                                 | Stimulates ACTH release                                |
| Gonadotropin - releasing hormone (GnRH)                                                 | Stimulates FSH & LH release                            |
| Prolactin - Releasing Peptide (PRP) <br><br>Dopamine (PRL- inhibiting hormone) (PIH)    | Stimulates PRL release<br><br>Inhibit PRL release      |
#### Control of hypothalamic hormones
Regulated by [[Control of hormone secretion#Negative Feedback control|Negative Feedback Loops]]

## Pineal 
- Tiny, pinecone-shaped structure located in center of brain
- Regulated by suprachiasmatic nucleus (SCN) - in hypothalamus above optic chiasm
	- synchronization of biological clock with environmental cues using **melanopsin**
Function - regulates circadian rhythm of brain via **melatonin**
