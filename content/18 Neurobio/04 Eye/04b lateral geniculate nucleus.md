# Parallel pathways
different pathways past the LGN along the brain
significance: magnocellular, parvo-interblob, and blob allow brain to **analyze different features of a visual scene simultaneously**, making image processing **faster, more detailed, and more efficient**.

# Non retinal inputs
only about 10% of inputs supplied are from [[02d Retina]], the rest mostly from [[01ba Anatomic Divisions of Brain#Brain stem|brain stem]] and primary visual cortex.
- role for this massive input has ==not been clearly identified.==
- *hypothesis:* “top–down” modulation from visual cortex to LGN gates, subsequent “bottom-up” input from LGN back to the cortex. E.g. ==selectively pay attention to portion of our visual field,== might be able to suppress inputs coming from outside the attended area.
