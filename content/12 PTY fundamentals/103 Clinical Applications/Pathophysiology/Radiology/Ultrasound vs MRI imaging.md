| [[Ultrasound imaging]]                    | [[MRI]]                            |
| ----------------------------------------- | ---------------------------------- |
| cheaper                                   | expensive                          |
| readily available                         | dedicated imaging centers          |
| fast                                      | slow                               |
| no contraindications                      | MR contraindications               |
| inter-operative dependence                | less operator dependence           |
| decreased sensitivity when limited by ROM | sensitivity not limited by ROM     |
| low sensitivity for non-cuff injuries     | high sensitivity for cuff injuries |

TLDR;
- MRI = gold standard especially for diagnostic imaging
- ultrasound = good for real time feedback and dynamic structures
