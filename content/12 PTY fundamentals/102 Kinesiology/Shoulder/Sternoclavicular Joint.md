## Structure
### Articular Surface  
	
Between Clavicle & Manubrium, form [[types of joints#^071d00|Saddle joint, 2 pairs]] [[Convex-Concave principle|convex-concave]] surfaces sitting perpendicular
	Allowing 2 planes of movement
![[Pasted image 20250126215129.png|center]]
### Ligaments

| Ligament                                                        | Proximal attachment | Distal attachment | Function | Features |
| --------------------------------------------------------------- | ------------------- | ----------------- | -------- | -------- |
| Anterior + Posterior sternoclavicular <br>joint ligaments (SCL) |                     |                   |          |          |
| Interclavicular ligament (ICL)                                  |                     |                   |          |          |
| Costoclavicular ligament (CCL)                                  |                     |                   |          |          |

Rule of thumb:
Move towards: slack e.g. PSL slack during elevation
Move away: taut e.g. CCL taut during elevation
