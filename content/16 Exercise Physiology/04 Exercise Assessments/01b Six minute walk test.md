Patient to walk as far as possible for 6 minutes (not about speed), then measure distance completed to estimate max walking speed
## Setup
- set 2 cones, 30m apart, measure using measuring/surveyor's wheel 
- Accuracy: mark out intervals of distance along the route 
#### Safety
Environment:
- set chairs along the route, if walking path narrow, can ask client/patient lean on walls
- ensure walking path clear from obstructions

Individual: 
- ensure patient in proper attire and footwear

## Execution
#### During
- Measure all patient vitals before commencing test
- Begin stopwatch, count number of laps completed using test template
- Every 1 min, say standardized motivation for patient
- Measure HR,SPO2 and RPE

#### End 
- Sit patient down at where they stopped
- Measure vitals, give rest
- check RPE
- check if anything prevented them from walking further


## Process Results
end distance x 10 = max walking speed
