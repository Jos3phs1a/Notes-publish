# Conduction related
## AV block
Impulses between atrial to ventricle blocked, cannot be transmitted 
### 1st deg AV block
- rarely serious, can be left untreated

Features:
constant prolonged PR intervals (>0.20s)

### 2nd deg AV block 

#### Mobitz type 1
- Treatment required only if AV block progress to [[#Mobitz type 2]] or [[#3rd deg ab block]]

Features:
increasing prolonged PR intervals 
QRS complex occasional disappearance
#### Mobitz type 2
![[Pasted image 20250728003211.png]]
- usually chronic
- CO ↓
- frequently progresses to [[#3rd deg AV block]]
- Requires intervention usually pacemaker

Features: 
constant prolonged PR intervals
occasional missing QRS

### 3rd deg AV block 
- occurs due to compromised CO
- requires intervention usually pacemaker

Features
independent P and QRS complexes
## Paced rhythm
Use of pacemaker induces rhythm in chamber that is visible on ECG

Features:
white vertical line at regular intervals in waveforms

### Single Chamber Atrial pacing
Features:
- Sharp pacing spike before P wave
